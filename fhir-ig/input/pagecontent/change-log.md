Below is a log of changes to the HL7 SMART Scheduling Links FHIR IG.

### Current
[Extends base SMART Scheduling Links Spec](https://github.com/smart-on-fhir/smart-scheduling-links/blob/master/specification.md) beyond the original use case to include general patient appointments.

#### New Resources
Added the following new resources
- HealthcareService
- Practitioner
- PractitionerRole

#### New Use Cases
- New vs. Existing patients
- Mobile clinics
- Virtual locations

### Release 1.0.0
Introduced [SMART Scheduling Links](https://github.com/smart-on-fhir/smart-scheduling-links/blob/master/specification.md) as a standards-based specification to enable patients to find appointment slots, follow a deep link, and complete a booking for COVID-19 immunizations.
